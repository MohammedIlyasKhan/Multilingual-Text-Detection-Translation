# Multilingual Text Detection & Translation

A desktop application built using **Python and CustomTkinter** that detects text from images in **multiple languages** (Kannada and English) and translates the detected text using **Google Cloud Translation API**.

This project focuses on **OCR (Optical Character Recognition)**, multilingual support, and real-time translation with a modern GUI.
